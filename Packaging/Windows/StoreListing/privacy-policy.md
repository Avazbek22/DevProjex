# Privacy Policy

DevProjex is a read-only desktop application.

What we read:
- Local folders and files you explicitly select.
- Git repository data that is cloned or fetched into the local cache you initiate.

What we do not do:
- We do not modify, delete, or write to your files.
- We do not send your files or contents to any server.
- We do not collect personal data or usage analytics.
- We do not perform self-updates.

Network access:
- Network is used only when you provide a Git URL for clone/fetch.
- No background network activity.

Contact:
- If you have questions, contact the publisher listed in the Microsoft Store listing.
