## 1) Ин чист ва барои чӣ
DevProjex контексти лоиҳаро зуд омода мекунад: сохтори ҷузвдонҳо/файлҳоро нишон медиҳад, имкон медиҳад элементҳоро бо қуттӣ интихоб кунед ва дарахт ё мундариҷаи файлҳоро ба буфер нусха бардоред. Барои code review, дастгирӣ, омӯзиш ва додани сохтори лоиҳа ба ёрдамчии AI қулай аст.

Муҳим: барнома танҳо дар режими хондан кор мекунад ва файлҳоро тағйир намедиҳад.

## 2) Дарахти лоиҳа
* Иконкаҳои навъи файл/ҷузвдон.
* Қуттиҳои интихоб дар гиреҳҳо.
* Кушодан/бастан.

Қуттиҳо: интихоб ба наслҳо татбиқ мешавад; ҳолати падар худкор ҳисоб мешавад.

Тугмаҳои зуд:
* Ctrl+E — кушодани ҳама
* Ctrl+W — бастани ҳама (реша кушода мемонад)

## 3) Файл
* Кушодани ҷузвдон… — кушодани ҷузвдони реша (Ctrl+O)
* Навсозӣ — бозхондан ва бозсозии дарахт (F5)

## 4) Git
Менюи Git танҳо барои лоиҳаҳое дастрас аст, ки тавассути DevProjex клон шудаанд (Git Clone). Барнома ба файлҳои шумо даст намерасонад — танҳо бо кеш кор мекунад.

Кеш дар ҷузвдони муваққатии система нигоҳ дошта мешавад. Дар Windows одатан:
`%TEMP%\DevProjex\RepoCache`

* Шоха — рӯйхати шохаҳои дурдастро нишон медиҳад; ҳангоми иваз кардан дарахт ба шохаи интихобшуда нав мешавад. Агар шоха аллакай маҳаллӣ бошад, ивазкунӣ бе шабака (бе навсозӣ) сурат мегирад. Агар набошад, он тавассути fetch бор карда шуда, маҳаллӣ сохта мешавад.
* Гирифтани навсозиҳо — танҳо шохаи ҷориро нав мекунад: fetch ва reset ба origin/<шоха>.

## 5) Нусха
### Нусхаи дарахт
Дарахти лоиҳаро нусха мебардорад.
Агар гиреҳҳои интихобшуда бошанд, дарахт аз рӯйи интихоб сохта мешавад; агар интихоб холӣ ё номувофиқ бошад — дарахти пурра нусха мешавад.
Тугмаҳо: Ctrl+Shift+C ва Ctrl+Alt+C.

### Нусхаи мундариҷа
Мундариҷаи файлҳоро нусха мекунад:
* агар файлҳо интихоб шуда бошанд — танҳо онҳо; вагарна — ҳамаи файлҳои лоиҳа;
* файлҳои ғайриматнӣ нодида гирифта мешаванд (аз ҷумла бинарӣ);
* барои файлҳои матнии холӣ роҳ ва сатри `[No Content, 0 bytes]` бароварда мешавад;
* барои файлҳое, ки танҳо аз фосилаҳо/гузариши сатрҳо иборатанд, роҳ ва сатри `[Whitespace, N bytes]` бароварда мешавад.
Тугма: Ctrl+Alt+V.

### Нусхаи дарахт + мундариҷа
Дарахт ва мундариҷаро якҷо мекунад:
* дарахт аз интихоб; агар интихоб холӣ ё номувофиқ бошад — дарахти пурра;
* мундариҷа мувофиқи қоидаҳои «Нусхаи мундариҷа».
Тугма: Ctrl+Shift+V.

## 6) Намоиш
### Масштаб
* Ctrl++ / Ctrl+= — калон кардан
* Ctrl+- — хурд кардан
* Ctrl+0 — баргардондан
* Ctrl + чархи муш — зум нарм

Диапазон: 6–28, баргардондан — 12.

### Режими ихчам
Интерфейсро зичтар мекунад: фосилаҳо дар дарахт ва танзимот хурдтар мешаванд.

## 7) Ҷустуҷӯ дар дарахт
Панели ҷустуҷӯро мекушояд, мувофиқатҳоро равшан мекунад ва навигатсия медиҳад.
* Ctrl+F — нишон/пинҳон
* Enter — натиҷаи навбатӣ
* Shift+Enter — қаблӣ
* Esc — пӯшидан

Ҷустуҷӯ танҳо шохаҳои заруриро мекушояд; ҳангоми тозакунии дархост дарахт ҷамъ мешавад (ғайр аз реша).

## 8) Филтр бо ном
Танҳо гиреҳҳои мувофиқ ва падарони онҳоро нишон медиҳад. Навсозӣ бо таъхир.
* Ctrl+Shift+N — филтрро нишон/пинҳон
* Esc — пӯшидани филтр

## 9) Танзимот
Дар тарафи рост рӯйхатҳо:
* Шрифти дарахт — бо Истифодаи танзимот татбиқ мешавад
* Нодида гирифтан
* Навъҳои файл
* Ҷузвдонҳои сатҳи боло

Дар ҳар рӯйхат тугмаи Ҳама (ҳамаро фаъол/хомӯш) мавҷуд аст.

Ctrl+P — панелро нишон/пинҳон.

## 10) Ignore ва Smart Ignore
### Қоидаҳои оддӣ
Хориҷ кардани артефактҳо ва «шум» (bin/obj, унсурҳои пинҳон, унсурҳои бо нуқта).

### Smart Ignore
Истисноҳои «ақлона» (ҷузвдонҳои VCS/IDE ва файлҳои партов). Қоидаҳои фронтенд танҳо ҳангоми ёфтани нишонаҳо фаъол мешаванд (package/lock файлҳо).

## 11) Дарахт чӣ тавр сохта мешавад
Натиҷа вобаста аст ба:
* интихобшуда ҷузвдонҳои сатҳи боло
* интихобшуда навъҳои файл
* қоидаҳои нодида гирифтан
* филтр фаъол

Агар дастрасӣ набошад, гиреҳ бо Дастрасӣ нест нишон дода мешавад.

## 12) Иконкаҳо
Тартиби интихоб:
1) навъи гиреҳ (ҷузвдон/файл)
2) номи дақиқ (агар қоида бошад)
3) васеъшавӣ
4) дар акси ҳол — номаълум

## 13) Мавзӯъ ва эффектҳо
Popover‑и мавзӯъ имкон медиҳад:
* мавзӯи равшан/торик
* Шаффоф / Mica / Acrylic
* параметрҳои интенсивият, blur, контраст ва сарҳад

Бандҳои бо [Beta] метавонанд байни версияҳо тағйир ёбанд.

## 14) Ёрӣ → Дар бораи барнома
Popover бо маълумот ва истиноди репозиторийро мекушояд.
Барқарор кардани танзимот (Ёрӣ → Барқарор кардани танзимот) танзимоти мавзӯъ ва эффектҳоро ба қиматҳои ибтидоӣ бармегардонад.
* Esc — пӯшидани popover

## 15) Забонҳо
Забон аз менюи Забон интихоб мешавад.

## 16) Эзоҳҳо
* Пешфарз мавзӯи торик ва эффектҳои шаффофият фаъол аст (агар дастгирӣ шавад).
* Ҳангоми нокифоя будани иҷозат, барнома метавонад бозоғозӣ бо ҳуқуқи баланд пешниҳод кунад; агар рад шавад — огоҳӣ нишон дода мешавад.
